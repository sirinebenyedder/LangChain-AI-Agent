## 🎯 Project Goal
To showcase a clean, functional implementation of a **JSONAgent** that manages:
* **Orchestration**: Using LangChain to bridge the Gemini model with specific tools.
* **Memory**: Persistent conversation history for contextual learning.
* **Tool-Calling**: Structured JSON-based interaction for pedagogical accuracy.

## 🛠️ Technical Stack
* **Framework**: LangChain
* **LLM**: Google Gemini 2.5 Flash
* **Interface**: Gradio
* **Logic**: Custom Python tools for Paradigm tutoring (OOP, Functional, etc.)
